# Python Beginner Course Series

Welcome to the official repository for the **Python Beginner Course Series**. This repository contains all the essential materials, including code examples, assignments, lecture slides, and capstone project documentation, to help you master the fundamentals of Python programming.

## 📂 Repository Structure

The repository is organized into the following directories:

* **`assignments/`**: Contains practice problems and hands-on exercises to reinforce learning after each module.
* **`capstones_docs/`**: Detailed documentation, requirements, and guidelines for the final capstone projects.
* **`code_examples/`**: Ready-to-run Python scripts and snippets demonstrated during the course sessions.
* **`slides/`**: Presentation materials (PDF/PPTX) covering the theoretical aspects of the curriculum.

## 🚀 Getting Started

To get the most out of this course, follow these steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/javiguerra777/python_beginner_course_files.git
    ```
2.  **Install Python**: Ensure you have Python 3.x installed on your machine. You can download it from [python.org](https://www.python.org/).
3.  **Explore the Modules**: Start with the `slides/` to understand the concepts, then dive into `code_examples/` to see them in action.

## 🛠️ Requirements

- Python 3.8 or higher
- A code editor (Visual Studio Code, PyCharm, or Thonny are recommended)
- Basic terminal/command prompt knowledge

## 📝 Course Content Overview

This course covers:
- Python Syntax and Variables
- Data Types (Strings, Integers, Lists, Dictionaries)
- Control Flow (If statements, For/While loops)
- Functions and Modules
- File I/O
- Basic Error Handling
